Hola lector, para empezar esto es un ejemplo super sencillo de como compilar un programa con OpenGL haciendo uso de CMake. El programa creado te servirá de preferencia en Visual Studio, debido a que a sido testeado su construcción en ese IDE.
En Visual Studio: Recuerda establecer OpenGLProyect como el Proyecto de inicio, de lo contrario te aparece el mensaje de "ALL_BUILD access denied"

In Visual Studio: Hi reader, to start this is a very simple example about how to compile a program with OpenGL using CMake. The created program will serve you as a preference in Visual Studio, due to its construction had been tested int that IDE.
Remember to set OpenGLProyect as startup Proyect, otherwise you will get the message "ALL_BUILD access denied"
